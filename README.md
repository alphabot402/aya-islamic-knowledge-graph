# AYA (آية) - Islamic Knowledge Graph

Explore authentic connections between Quran verses and hadiths through beautiful visualization.

## Quick Start

1. Install dependencies:
```
   npm install
   cd app
   npm install
```

2. Start development server:
```
   cd app
   npm run dev
```

3. Open http://localhost:3000

## Project Structure

- app/ - Next.js application
- data/ - Verified JSON data
- pipeline/ - Data processing scripts
- docs/ - Documentation
- tests/ - Validation
